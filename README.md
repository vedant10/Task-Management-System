# Task-Management-System
Angular 17
Node.Js
SQL

////////////////////////////////////////////////////////////////
Project setup
Node.js Server
cd node-express-sequelize-mysql
Run nodemon server.js
//////////////////////////////////////////////////////////////////
Angular Client
cd angular-17-client
Run ng serve --port 8081. Navigate to http://localhost:8081/.
